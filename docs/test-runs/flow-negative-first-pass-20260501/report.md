## 验证结果

| 校验项 | 实际结果 |
| --- | --- |
| 未派单订单直接审核通过 | 返回 `400 当前订单状态不能审核` |
| 未派单订单直接审核退回 | 返回 `400 当前订单状态不能审核` |
| 未派单订单派单给临时设计师 | 成功，状态变为 `未完工` |
| 未完工订单重复调用首次派单接口 | 返回 `400 只有未派单订单可以派单` |
| `shejishi` 查询他人订单 | 设计师列表为空 |
| `shejishi` 直接完工他人订单 | 返回 `403 无权限` |
| 订单所属临时设计师提交完工 | 成功，状态变为 `待审核` |
| 待审核订单再次提交完工 | 返回 `400 当前订单状态不能提交` |
| 清理订单、商户、临时用户 | 全部通过且查询无残留 |
